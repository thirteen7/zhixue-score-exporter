# 智学网学生成绩导出脚本

一个用于在智学网“校级报告 -> 学生成绩”页面导出学生成绩的浏览器控制台脚本。

脚本会自动读取当前页面的“全部班级”数据，翻页抓取所有学生成绩，并按“班级”列自动拆分为多个 Excel 工作表。它不写死年级和班级名称，因此切到三年级、四年级、五年级等页面后都可以直接使用。

## 功能

- 自动切换到“全部班级”并抓取当前年级全部学生成绩。
- 自动翻页，直到没有下一页。
- 按班级拆分工作表，例如 `三1班`、`三2班`、`三3班`、`三4班`。
- 工作表按班级自然排序，而不是按成绩榜首次出现顺序排序。
- 保留 `全部班级` 和 `汇总` 工作表。
- 自动识别页面上的科目列，适配不同年级的科目变化。
- 导出为 `.xlsx` 文件。

## 使用方法

1. 打开智学网教师端，进入目标考试的“校级报告 -> 学生成绩”页面。
2. 确认页面上能看到成绩表格。
3. 打开浏览器开发者工具。
4. 切换到 `Console` 控制台。
5. 复制 [outputs/zhixue-export-scores.js](outputs/zhixue-export-scores.js) 的全部内容并粘贴到控制台。
6. 按 `Enter` 运行。
7. 等待控制台提示“导出完成”，浏览器会下载 `智学网学生成绩导出.xlsx`。

## 下载其他年级

先在智学网页面切换到对应年级或对应考试，再运行同一个脚本即可。脚本会根据当前页面上的真实数据自动识别班级和科目。

## 注意事项

- 运行脚本前请保持账号已登录。
- 如果浏览器提示是否允许粘贴到控制台，需要按浏览器提示确认。
- 如果提示 `SheetJS 加载失败`，通常是网络或 CDN 被拦截导致，可以把 SheetJS 改成本地引入版本。
- 请不要把导出的学生成绩 Excel 文件提交到公开仓库。
